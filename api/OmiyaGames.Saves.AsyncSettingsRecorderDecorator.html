    Class AsyncSettingsRecorderDecorator

    A decorator that implements a couple of 

    AsyncSettingsRecorder
    methods by wrapping a couple of methods with other methods. Extending this class should reduce the amount of work necessary to implement an 
    AsyncSettingsRecorder
    .
